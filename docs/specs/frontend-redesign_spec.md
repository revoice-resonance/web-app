# Specification: Frontend Redesign — Match project-resonance.net

> 中文界面，保留英文技术术语。

## Overview

重新设计前端，使其在视觉和功能上与 https://www.project-resonance.net/ 完全一致，同时保留现有的 Whisper ASR / CosyVoice TTS 后端集成。

**目标网站实际页面结构（来自浏览器探索）：**
| 路由 | 名称 | 功能 |
|------|------|------|
| `/` | 使用 | 主录音 + ASR 识别 + TTS 朗读 |
| `/settings` | 设置 | 辅助功能、ASR 配置、识别参数、TTS 参数 |
| `/phrases` | 词表管理 | 100 个短语，10 个分类，搜索/编辑/导入导出 |
| `/training` | 录音训练 | 逐短语录音，进度追踪，回放 |
| 欢迎页 | 引导流程 | 首次使用引导 |

**当前仓库已有这些页面，只需要视觉对齐。Whisper ASR / CosyVoice TTS 后端已就绪，不需要修改。**

---

## Requirements

### R1 — 主"使用"页面视觉重设计

**AC:**
- 暗色海军蓝头部，含"共鸣"品牌 + "PROJECT RESONANCE"副标题
- 导航标签："使用" | "设置"
- 渐变文字标题"语音识别"
- 流程指示器药丸："录音 → 识别 → 存音色/朗读"
- 键盘提示："按 空格 开始录音"（kbd 样式）
- 大圆形录音按钮居中，麦克风图标
- 录音状态：红色脉冲计时器，"点击或按 空格 停止录音"
- 处理状态：旋转动画，"正在识别语音..."
- 结果状态：识别文本 + 候选词选择（1-3）+ 操作按钮（朗读/复制/再说一次）
- 错误状态：友好的中文错误提示 + 重试操作
- 页脚：构建时间戳
- 保留所有现有功能（录音、ASR、TTS、复制、重置、微信桥接）

### R2 — 设置页面视觉重设计

**AC:**
- "设置"标题 + "调整识别、语音与辅助功能参数"描述
- 快速访问卡片："词表管理"和"录音训练"（链接到 /phrases 和 /training）
- **辅助功能部分**：停留控制（5 级）、防误触（4 级）、大号点击区域、字体大小（5 级）、减弱动态效果、增强对比度、始终显示焦点
- **语音识别配置**：Whisper 状态指示器
- **识别参数**：Top-K、置信度阈值、最大模板数滑块
- **TTS 参数**：语速、音量、音高滑块 + 语音选择器
- 各部分的恢复默认按钮

### R3 — 词表管理页面视觉重设计

**AC:**
- "词表管理"标题 + "添加、编辑、导入导出短语"描述
- 搜索栏
- 100 个短语，10 个分类（中文标签）
- 分类标题带彩色图标
- 每个短语：文本、开关、编辑按钮
- 添加短语、导入/导出功能
- 视觉设计与目标一致

### R4 — 录音训练页面视觉重设计

**AC:**
- "录音训练"标题 + "录制语音样本用于个性化识别"描述
- 进度追踪："X/100 已训练"徽章
- 筛选标签："录音训练" / "完成"
- 每个短语的录音状态、录音次数、录音按钮
- 已录音样本的回放功能
- 视觉设计与目标一致

### R5 — 欢迎/引导流程

**AC:**
- 多步骤引导流程，与目标一致
- 跳过选项
- 完成后在 localStorage 中存储标记

### R6 — 键盘快捷键模态框

**AC:**
- "?" 键触发
- 列出所有快捷键：空格（录音）、R（重置）、T（朗读）、C（复制）、Escape（取消）、1-3（候选词）、?（帮助）
- Esc 关闭

### R7 — 全局导航和布局

**AC:**
- 暗色海军蓝头部，含品牌 + 导航标签
- "使用"→ `/`，"设置"→ `/settings`
- 头部中的键盘快捷键帮助按钮
- "跳到主要内容"无障碍跳转链接
- 移动端响应式
- 设置页面包含指向 /phrases 和 /training 的快速访问卡片

### R8 — 辅助功能特性

**AC:**
- 停留控制：悬停点击，可配置延迟
- 防误触：在配置的时间窗口内防抖快速点击
- 大号点击区域：切换开关，增大所有交互元素
- 字体缩放：5 级，影响所有文本
- 减弱动态效果：禁用动画
- 增强对比度：提高对比度
- 始终显示焦点：鼠标点击时也显示焦点环
- 所有设置持久化到 localStorage，立即生效

## Dependencies
保留的钩子（无变更）：useCloudASR、useCloudTTS、useAudioRecorder、useKeyboardShortcuts、useAccessibility、useAppData、useAuth、useWechatBridge
保留的组件（可能需要重新样式化）：AudioRecorderButton、ASRStreamingResult、ASRErrorBanner、AudioSpectrum、TTSButton

## Out of Scope
后端/Worker 变更、Go 服务器、数据库、新的 ASR/TTS 提供商、语音克隆 UI、身份验证变更、新短语、国际化

## Assumptions
Whisper / CosyVoice API 密钥已配置且可用，目标网站截图代表期望的最终状态，现有钩子稳定，Inter 字体，移动端响应式，保留微信集成
